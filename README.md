# lahu

这是一个低代码平台开源项目

## packages

- `engine`，项目画布引擎
- `transfomer`，项目 `Schema` 转译
- `mock-server`，项目 `API` 接口服务和远程（本地）部署服务

## 命令

- `npm run install`，为所有项目安装依赖，**注意**：`npm install` 无效，只能安装当前项目
- `npm run dev`，运行项目
- `npm run server`，启动本地接口 // 暂时为空，无需开启
